Please follow the below steps inorder to get res101.mat and att_splits.mat files to run this project.
    Initailly, run mataw.py file in order to generate {dataset}_img/label.csv files.
    Run mat_create.m file in matlab to get res101.mat file.
    Then, execute dataset_create.py file in order to obtain att_splits.mat file.
Place the res101.mat and att_splits.mat files in corresponding dataset directory, i.e., ZS-ViDAT/data/xlsa17/data/{DATASET}/res101.mat, att_splits.mat
